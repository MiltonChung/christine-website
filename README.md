# Christine's Personal Website

This website was made purely with HTML/CSS and some JavaScript for the go to top button in mobile view. It was created based on her requested design and a theme of minimalism. We went through many iterations and she gave me constant feedback and updates on features/pages that she wants. I followed my client's instructions very carefully, from the fonts she chose to the exact margins and placements of the texts and buttons. On top of that, I made the website responsive so it is user friendly on all screensizes and platforms.
